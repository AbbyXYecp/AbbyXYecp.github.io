字符串操作方法汇总-JavaScript

发表于 2020-10-22 | 分类于 JS | 次

字符串操作汇总

  • let num = 56

    字符串的转换

  • num.toString()
  • String(num)
  • let a = “I Love China”

字符串的长度

  • a.length

指定位置字符

  • a.charAt(index)

指定位置字符Unicode编码

  • a.charCodeAt(index)

连接字符串

  • a.concat(b)
  • let b = “ So Much !”

检索字符串

  • a.indexOf() 返回检索字符首次出现的位置,不存在则返回-1,对大小写敏感
  • let a = “I Love China, I Love China !”
  • a.lastIndexOf() 返回检索字符最后出现的位置,不存在则返回-1,对大小写敏感
  • let a = “I Love China, I Love China !”
  • a.match() 返回检索字符串内指定的值，或找到一个或多个正则表达式的匹配,结果返回数组
  • 数组拥有一个额外的 input 属性，该属性包含原始字符串。
  • 数组还拥有一个 index 属性，该属性表示匹配结果在被字符串中的索引（以0开始）。
  • 如果正则表达式包含 g 标志，则该方法返回匹配字符串的数组。
  • let a = “I Love China, I Love China !”
  • exec(a)
  • a.search() 返回检索字符首次出现的位置, 不存在则返回-1,对大小写敏感

字符串替换

  • a.replace() 用一些字符替换另一些字符，或替换一个与正则表达式匹配的子串

字符串分割

  • a.slice(start, end) 使用 start（包含） 和 end（不包含）参数来指定字符串提取的部分
  • 如果是负数，则该参数规定的是从字符串的尾部开始算起的位置。也就是说，-1 指字符串的最后一个字符，-2 指倒数第二个字符，以此类推。
  • a.substring(start, end) 使用 start（包含） 和 end（不包含）参数来指定字符串提取的部分,参数只能非负整数
  • a.substr(start, end) 从起始位置开始截取，结束位置为第二个参数截取的字符串最大长度

字符串大小写转换

  • a.toLowerCase() 大->小
  • a.toUpperCase() 小->大

字符串去空格

  • a.trim() 删除字符串前后的空格
